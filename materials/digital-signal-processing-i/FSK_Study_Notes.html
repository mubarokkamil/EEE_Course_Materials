01 ──

What is FSK? (The Big Idea)

Before understanding FSK, you need to know two basics: what is a carrier wave, and what is modulation.

📡 Carrier Wave

A carrier wave is a simple, steady sine wave (a smooth up-and-down wave) that your transmitter sends out. It has three properties you can change: Amplitude (height), Frequency (how fast it oscillates), and Phase (starting position). These are your "knobs" for encoding data.

🔧 Modulation

Modulation is the art of changing one of those knobs to carry information. You have digital data (0s and 1s), but the channel sends waves. Modulation bridges that gap — it maps your bits onto wave properties.

So then, Frequency Shift Keying (FSK) is a digital modulation technique where you encode data by switching the frequency of the carrier wave. Every time your data bit changes, the frequency of the wave shifts to a different value.

💡 Real-World Analogy

Imagine a musician playing notes to send a secret message. If the note is high-pitched (high frequency), that means binary "1". If the note is low-pitched (low frequency), that means binary "0". The receiver just listens to which pitch is being played and decodes the message. FSK works exactly like this — just with radio waves instead of musical notes.

02 ──

How FSK Works — Step by Step

Let's trace the entire process of transmitting the binary data 1 0 1 1 0 using FSK.

  • Choose your two frequencies. You decide: f₁ = 2000 Hz (for bit 1) and f₀ = 1000 Hz (for bit 0). These are your two "symbols."
  • Look at your first bit. The first bit is 1. So the transmitter outputs a sine wave at 2000 Hz for the duration of one bit period.
  • Move to the next bit. The next bit is 0. The transmitter instantly switches the frequency to 1000 Hz and holds it there for one bit period.
  • Continue for each bit. This pattern continues — bit 1 → high freq, bit 0 → low freq — for each remaining bit in the stream. The result is a wave whose frequency hops between values.
  • The receiver detects the frequency. On the other end, a frequency detector (like a discriminator circuit or PLL) reads which frequency is present at each moment and reconstructs the original bit stream perfectly.

🔑 Key Insight: Only Frequency Changes, Not Amplitude

Unlike AM (Amplitude Modulation), in FSK the amplitude stays the same throughout. This is a huge advantage because noise and interference usually affect amplitude, not frequency. So FSK is naturally more resistant to noise.

04 ──

Types of FSK

FSK comes in several flavors depending on how many frequencies are used and how cleanly the transitions happen.

Binary FSK (BFSK or 2-FSK) — The Basic Version

Uses exactly two frequencies: one for bit 0 and one for bit 1. This is the simplest form of FSK. Each signal state carries exactly 1 bit of information. Easy to implement, widely taught in textbooks.

Bit 1 → f₁ = 2000 Hz  |  Bit 0 → f₀ = 1000 Hz

M-ary FSK (MFSK) — More Frequencies, More Data

Instead of just 2 frequencies, MFSK uses M different frequencies. Each frequency represents a group of bits (called a symbol). For example, 4-FSK uses 4 frequencies and can encode 2 bits per symbol, while 8-FSK uses 8 frequencies and encodes 3 bits per symbol.

4-FSK: 00→f₁, 01→f₂, 10→f₃, 11→f₄   (2 bits per symbol)

Continuous Phase FSK (CPFSK) — Smoother Transitions

In basic FSK, when the frequency switches, the wave can have an abrupt "jump" in phase — like a hiccup. CPFSK ensures the wave transitions smoothly without any phase discontinuity. This uses less bandwidth and causes less interference.

Minimum Shift Keying (MSK) — The Optimal FSK

MSK is a special case of CPFSK where the frequency difference between f₁ and f₀ is set to the minimum possible value that still allows the two frequencies to be mathematically distinguishable (orthogonal). This gives the most bandwidth-efficient form of FSK. Used in GSM mobile networks.

Gaussian MSK (GMSK) — Smooth and Spectrally Efficient

GMSK takes MSK and adds a Gaussian filter to the data before modulating. This pre-filters the bit transitions, making the frequency changes even smoother, which reduces the bandwidth further. Used in GSM 2G cellular networks worldwide.

05 ──

Key Formulas — Explained Simply

Don't just memorize — understand what each formula is saying.

FSK Signal Expression
s(t) = A · cos(2π · fᵢ · t)

where fᵢ = f₁ if bit = 1, or f₀ if bit = 0

This just says: "At any moment in time t, the FSK signal is a cosine wave with amplitude A and whichever frequency corresponds to the current bit." Nothing magical — just math for the wave.

Frequency Deviation (Δf)
Δf = (f₁ − f₀) / 2

Also written as: f₁ = fc + Δf and f₀ = fc − Δf

The deviation tells you how far each frequency is from the center. If fc = 1500 Hz and Δf = 500 Hz, then f₁ = 2000 Hz and f₀ = 1000 Hz. It measures the "spread" between your two frequencies.

FSK Bandwidth (Carson's Rule Approximation)
BW = 2 · (Δf + fₘ)

where fₘ = maximum frequency of the modulating signal (≈ bit rate / 2)

The bandwidth your FSK signal occupies depends on both the frequency deviation AND the bit rate. Larger deviation or higher bit rate = more bandwidth consumed.

Modulation Index (h) — The Deviation Ratio
h = (f₁ − f₀) / (bit rate) = 2·Δf / Rb

The modulation index tells you how much the frequency is shifting relative to the bit rate. For MSK, h = 0.5 (the minimum that allows the two tones to be orthogonal — perfectly distinguishable without interference between them).

Bit Error Rate (BER) for Coherent BFSK
Pb = Q(√(Eb / N₀))

For Non-Coherent BFSK: Pb = (1/2) · exp(−Eb / 2N₀)

BER tells you what fraction of bits arrive wrong. Eb/N₀ is your signal quality (energy per bit divided by noise). Higher Eb/N₀ = fewer errors. Coherent detection (which knows the exact phase) performs better than non-coherent.

Bandwidth — How Much Spectrum Does FSK Use?

Bandwidth is one of the most important concepts — it determines how much of the radio spectrum your signal takes up, and spectrum is scarce and expensive.

The Intuition Behind FSK Bandwidth

When you switch between two frequencies rapidly (high bit rate), you create "spectral splatter" — energy spreads beyond your two tones. The total bandwidth = the separation between frequencies + spreading due to bit-rate transitions.

FSK Bandwidth Diagram (Frequency Domain View)

Practical Tradeoff

Larger Δf (more separation) → easier to distinguish f₁ from f₀, fewer errors, but more bandwidth used.

Smaller Δf (less separation) → less bandwidth used, but harder to tell the frequencies apart → more errors. MSK sits at the sweet spot — smallest Δf that still works perfectly.

08 ──

Advantages & Disadvantages of FSK

Advantages
  • Highly resistant to noise and interference — amplitude stays constant, and noise rarely changes frequency
  • Easy to implement in hardware — simple frequency switching circuits
  • Constant envelope (constant amplitude) makes it suitable for non-linear amplifiers without distortion
  • Better error performance than ASK in noisy channels
  • Good performance over telephone channels — historically used in modems
  • No information in amplitude — won't be affected by amplitude fading
Disadvantages
  • Uses more bandwidth than ASK or PSK — needs space for multiple frequency tones
  • Lower spectral efficiency than PSK/QAM — not ideal for high-data-rate systems
  • Abrupt frequency transitions in simple FSK create spectral splatter (extra bandwidth)
  • Not as spectrally efficient as QPSK or higher-order QAM used in modern systems
  • Multi-tone FSK (MFSK) becomes complex at high M values
  • Susceptible to multipath fading in mobile environments (frequency-selective fading)

Exam Tips & Common Questions

These are the things examiners love to test. Make sure you can answer all of these.

Q1
What property of the carrier wave changes in FSK?
Answer: Only the frequency. Amplitude and phase remain constant. This is the definition.
Q2
What is the modulation index of MSK and why?
Answer: h = 0.5. This is the minimum value that makes f₁ and f₀ orthogonal (mathematically independent). At h = 0.5, the two tones don't interfere, giving perfect detection.
Q3
Why is FSK more noise-resistant than ASK?
Answer: Noise primarily affects the amplitude of a signal. Since FSK encodes information in frequency (not amplitude), amplitude noise has no effect on the data — the receiver detects frequency, not voltage level.
Q4
What is the bandwidth of a BFSK signal with bit rate Rb and frequency deviation Δf?
Answer: BW ≈ 2(Δf + Rb/2) = 2Δf + Rb. This is Carson's approximation.
Q5
Difference between coherent and non-coherent detection of FSK?
Answer: Coherent detection knows the exact phase of the signal (better BER, more complex). Non-coherent doesn't need phase info (simpler, slightly worse BER). Non-coherent is often used because phase recovery is difficult in FSK.
Q6
What is CPFSK and why is it better than conventional FSK?
Answer: CPFSK = Continuous Phase FSK. It maintains phase continuity at frequency transitions. Conventional FSK can have phase jumps that cause spectral splatter (wasted bandwidth). CPFSK eliminates this, using less bandwidth.
Q7
How many bits does M-ary FSK carry per symbol?
Answer: log₂(M) bits per symbol. 4-FSK carries 2 bits, 8-FSK carries 3 bits, 16-FSK carries 4 bits, etc.
Q8
Which mobile standard uses GMSK and why?
Answer: GSM (2G) uses GMSK. Because GMSK is a spectrally efficient form of FSK with constant amplitude (good for battery life), smooth phase transitions, and solid noise performance.

The One-Page Summary

FSK = Frequency Shift Keying — a digital modulation technique that encodes binary data by switching the carrier wave's frequency between two (or more) preset values.
▸ Bit 1 → high frequency (f₁)  |  Bit 0 → low frequency (f₀)  |  Amplitude stays constant
▸ Binary FSK (BFSK) uses 2 frequencies, 1 bit/symbol. M-FSK uses M frequencies, log₂M bits/symbol
▸ CPFSK adds smooth phase transitions. MSK uses minimum orthogonal separation (h=0.5). GMSK filters MSK for even less bandwidth
▸ Bandwidth ≈ 2(Δf + Rb/2). Modulation index h = 2Δf / Rb
▸ Advantages: Noise resistant, constant amplitude, simple. Disadvantages: More bandwidth than PSK, less spectrally efficient
▸ Used in: Caller ID, GSM/2G, amateur radio RTTY, telemetry, SCADA, power line communications
